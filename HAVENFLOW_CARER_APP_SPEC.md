# Haven Flow Carer App — as-built reference

## Overview

This document name is kept for historical links. The **accurate description of the running product** (backend, API, web, mobile) is maintained in:

- **`docs/CAPABILITIES.md`** — module-by-module shipped behaviour  
- **`README.md`** — role-based overview in the monorepo root  
- **`backend/API_IMPLEMENTATION.md`** — `/api/v1` route catalogue  
- **`backend/README.md`** — legacy `/api` routes and server configuration  

## Key capabilities

All items that appear in those files are limited to **implemented** end-to-end flows (server routes plus UI where applicable).

## How it works

Clone the repo, follow **`QUICK_SETUP.md`**, run backend migrations and seed, start web and mobile, and exercise flows listed in **`docs/CAPABILITIES.md`**.
